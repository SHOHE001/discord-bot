# Manga Update Plugin

## 概要
ワンパンマン（原作）のサイト（http://galaxyheavyblow.web.fc2.com/）の更新を検知し、専用チャンネルに通知するプラグイン。

## 定期実行
- cron式: `0 8,20 * * *`（毎日 8:00, 20:00 JST）
- 動作: サイトをスクレイピングして更新テキスト（例:「〇〇年〇月〇日 〇〇話更新」）を取得。前回取得したテキストから変化があれば更新されたとみなし、`manga-update` チャンネルに通知します。

## 環境変数
不要。

## 依存パッケージ
- `cheerio`
- `node-fetch`

## チャンネル
自動で `manga-update` チャンネルを作成します。
